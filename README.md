## FAQ

- 네이버 지도가 있는데 굳이 카카오맵을 활용한 이유가 있는가?
- 카카오가 평점이 짜서 선택한 것입니다. 대체로 카카오에서 리뷰가 많으면서 평점이 높으면 신뢰할 수 있습니다. 보통 3.5점 만 넘어도 맛에는 하자가 없는 식당입니다.

- 굳이 로컬에서 실행하는 CLI로 만든 이유가 있는가?
- 실제 웹 서비스가 되면 클라우드에 올리거나 아니면 홈랩을 구성해야 합니다. 이 앱으로 돈을 벌 생각이 없습니다. 또 식사라는 경험은 개인적입니다. 개인적인 데이터는 본인이 관리해야 한다고 봅니다.
